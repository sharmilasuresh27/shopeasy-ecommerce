# ShopEasy - AI Powered E-Commerce Platform

## Overview

ShopEasy is a full-stack MERN E-Commerce platform built with modern web technologies. The application provides a complete online shopping experience with advanced features such as AI-powered product assistance, order tracking, wishlist management, and an admin dashboard.

## Features

### User Features
- User Authentication & Authorization
- Product Search & Category Filters
- Shopping Cart Management
- Wishlist Functionality
- Real-Time Price Drop Notifications
- Order Tracking System
- Responsive Design

### Admin Features
- Product Management (Create, Read, Update, Delete)
- Inventory Management
- Revenue Analytics Dashboard
- Order Status Management
- Customer Order Monitoring

### AI Features
- AI Shopping Assistant
- Smart Product Recommendations
- Instant Product Filtering

## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT (JSON Web Token)

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/shopeasy.git
```

2. Install dependencies

```bash
npm install
```

3. Configure environment variables

Create a `.env` file and add:

```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

4. Start the application

```bash
npm start
```
